# Currency Converter (USD ⇄ BRL)

A simple and lightweight currency converter built with Python and Tkinter.  
This application converts Brazilian Real (BRL) to US Dollar (USD) using real-time exchange rates from an external API.

---

## Features

- Real-time BRL to USD conversion  
- Simple graphical interface  
- Fast and lightweight  
- Input validation  

---

## Technologies Used

- Python  
- Tkinter  
- Requests
